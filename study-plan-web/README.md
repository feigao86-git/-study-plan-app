# 学习计划 Web 版 - 小猫猫粮版

初中生学习计划跟进App的网页版演示。采用小猫+猫粮的养成机制，完成任务获得猫粮来投喂小猫。

## 🎮 核心玩法

### 猫咪养成系统
- **一只可爱的小猫** (`🐱`) 作为学习伙伴
- **积分兑换猫粮**: 10积分 = 1碗猫粮 (`🥣`)
- **投喂互动**: 消耗猫粮投喂小猫，保持它的好心情
- **状态变化**:
  - 😸 很开心 (猫粮≥5)
  - 😺 还不错 (猫粮≥2)
  - 😐 一般 (猫粮≥1)
  - 😿 饿了 (猫粮=0)

### 任务系统
- **添加任务**: 选择学科、设置提醒时间
- **完成任务**: 获得10积分 + 1碗猫粮
- **猫粮动画**: 完成任务时弹出猫粮碗动画 (`🥣 × N`)

## ✨ 功能特性

### 已实现
- ✅ **猫咪展示**: 首页显示小猫和猫粮库存
- ✅ **猫粮系统**: 10积分兑换1碗猫粮

## 🎯 使用说明

### 1. 完成任务获得猫粮
```
点击任务 [完成] 按钮
↓
弹出猫粮动画: 🥣 × 1
↓
获得 10 积分
↓
积分自动兑换为 1 碗猫粮
```

### 2. 投喂小猫
1. 点击底部导航 [宠物]
2. 在"投喂"标签页点击 [🐱 投喂小猫咪]
3. 消耗 1 碗猫粮
4. 小猫会更开心！

### 3. 购买猫粮
1. 在"宠物"页面切换到"买猫粮"标签
2. 选择购买数量
3. 点击确认购买 (10积分/碗)

## 🐱 猫咪状态

| 猫粮数量 | 表情 | 状态 | 说明 |
|---------|------|------|------|
| ≥5 | 😸 | 很开心 | 小猫咪很开心，主人真棒！ |
| 2-4 | 😺 | 还不错 | 小猫咪感觉还可以 |
| 1 | 😐 | 有点饿 | 小猫咪饿了 |
| 0 | 😿 | 非常饿 | 小猫咪非常饿，快投喂吧！ |

## 💡 设计理念

### 为什么选择小猫+猫粮？
1. **简单直观**: 猫粮数量就是硬通货，一目了然
2. **即时反馈**: 完成任务立即看到猫粮增加
3. **情感连接**: 小猫会饿，需要持续投喂
4. **积分消耗**: 多余的积分可以大量购买猫粮

### 兑换比例
- **10 积分 = 1 碗猫粮**
- 完成 1 个任务 = 10 积分 = 1 碗猫粮
- 连续打卡奖励 50 积分 = 5 碗猫粮
